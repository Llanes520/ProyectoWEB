\# 🗄️ Copia de seguridad de MongoDB



Esta carpeta contiene una copia de la base de datos \*\*MyVeterinaria\*\*.



\### Restaurar la base de datos local



1\. Asegúrate de tener MongoDB y las Database Tools instaladas.

2\. Abre una terminal en la raíz del proyecto.

3\. Ejecuta el siguiente comando:



```bash

mongorestore --db MiVeterinaria .\database\backup_mongo\MyVeterinaria
